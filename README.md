# HuaweiRouterSupport2
Aplicación móvil desarrollada en SwiftUI que permite registrar, consultar, actualizar y eliminar reportes de diagnóstico de routers Huawei. La app se conecta a Firebase para almacenar los datos de manera segura y en tiempo real. Incluye funcionalidades CRUD completas y pruebas de seguridad aplicadas sobre la comunicación y el código
